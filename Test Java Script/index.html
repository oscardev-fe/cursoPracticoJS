  • ¿Qué es una variable y para qué sirve?

    Son espacio en memoria para almacenar información, dependiendo de los tipos y estructuras de datos que soporte nuestro lenguaje.

  • ¿Cuál es la diferencia entre declarar e inicializar una variable?

    Declarar es cuando le decimos a JavaScript que vamos a crear una variable, con el nombre tal.
    Iniciablizar/reinicializar es asignarle una valor a esa variable.

  • ¿Cuál es la diferencia entre sumar números y concatenar strings?

    El operador que nos permite sumar o concatenar es +. Este operador nos permite sumar números cuando lo usamos con números y concatenar cuando son strings.
